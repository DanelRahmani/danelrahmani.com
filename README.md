# My personal Website
This is the code for my personal website, it is a modified version of Bartosz Jaorcki's Website. I have added some features that I personally appreciate like a Resume download button, Email Button and Threads Button on the About Page. 
I also added some nice social media sharing icons on the blogposts. Further adjustments include the colour scheme, image orientation and the addition of a spotify Embed.
Feel free to use my code, attribution is appreciated.
